# Slash commands — keyforge-protocol

Custom slash commands for this repo go here. One file per command, format:

```markdown
---
description: What this command does
---

Prompt template. Use $ARGUMENTS to receive args from the slash invocation.
```

Invoke from chat with `/<filename-without-md>`.

Empty for now — add commands as recurring shortcuts emerge (e.g. `/new-message`, `/regen-types`).
